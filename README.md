# Pipewire sine wave generator
Taken from Pipewire tutorial [Part 4: Playing A Tone](https://docs.pipewire.org/page_tutorial4.html).

## Description
`pw-sine` is a simple tool for generating a sine wave using Pipewire.

## Options
- `-h` | `--help`
    - show help
- `--version`
    - show version
- `-f` | `--frequency`
    - sine wave frequency
- `-v` | `--volume`
    - volume (0.0-1.0)
